# <a name="speech-resources-and-samples"></a>语音资源和示例

使用 Windows 10 IoT Core 提供的识别和合成功能，您可以为您的设备生成一个完整的语音界面。

## <a name="whats-available-and-useful-links"></a>可用和有用的链接

下面是 Windows 10 IoT Core 上可用的服务和 API 列表：

* 语音识别（SR）
* 综合文本转语音（TTS）
* 连续听写
