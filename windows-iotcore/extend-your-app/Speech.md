# <a name="speech-resources-and-samples"></a>语音资源和示例

使用 Windows 10 IoT 核心版 提供的识别和合成功能，你可以为设备构建完整的语音接口。

## <a name="whats-available-and-useful-links"></a>可用和有用的链接

下面是可用服务和 api on Windows 10 IoT 核心版：

* 语音识别 (SR) 
* 合成文本转语音 (TTS) 
* 连续听写
